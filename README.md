### Usage
Basic steps for unix-likes:
```bash
git clone https://github.com/TheKrafter/f-bot
cd f-bot/
cp example.config.yml config.yml
# Now get your bot token
nano config.yml # put ur bot token where it says
python3 bot.py
```
